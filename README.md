A small python application for evaluating algebraic expressions.

## Install it from PyPI

```bash
pip install graphalith
```

## Usage

```py
from graphalith import BaseClass
from graphalith import base_function

BaseClass().base_method()
base_function()
```

```bash
$ python -m graphalith
#or
$ graphalith
```
